# last_input

![Image](https://github.com/user-attachments/assets/8a94457b-ab5f-4a97-8f22-2ca547d31a35)

### description

Windowsが認識しているPC接続のキーボードやマウスなどの  
入力デバイスが最後に入力されてからの経過時間を表示します。  

WindowsにはPC画面オフやスリープまでの経過時間を表示する機能がないので作成しました。  

以下の時間確認などに利用できます。

・キーボードやマウスを操作してからの時間  
・PCがスリープ機能で電源オフするまでの時間  
・離席時間など未操作から復帰するまでの時間  
・Windowsスリープ機能の設定時間の確認  
・PCのスリープ機能が働かないことの確認  

詳細は当方のブログ記事などで確認して下さい。  
https://www.enoanoarts.com/2022/04/app-release-lastinput.html  
Windowsの入力経過時間を表示するアプリ「Last Input」を公開しました
